# F1 Driver Archetype Clustering

**DMW 2301 – Data Mining and Wrangling | Mini Project 2**

## Overview

This project applies unsupervised learning to the careers of 77 Formula 1 drivers (2000–2024) to discover **driver performance archetypes**. Eight features are engineered from race results and qualifying data, reduced to three principal components via PCA, then clustered using K-Means, Ward's Hierarchical Clustering, and DBSCAN.

## Setup

```bash
# 1. Create and activate the conda environment
conda env create -f environment.yml
conda activate bsdsba2028-dmw-2301-mp2-lt-x

# 2. Place the five Ergast CSV files in data/raw/
#    results.csv, drivers.csv, races.csv, qualifying.csv, status.csv

# 3. Build the combined dataset
python data.py

# 4. Run unit tests
pytest tests/ -v

# 5. Launch the report notebook
jupyter lab report.ipynb
```
